# Multimodal Iterative Learning System

A simple DL project for patient behavior monitoring using real-time fusion of Emotion, Environment, Gesture, and Speech.

### Key Features:
- **Iterative Continual Learning**: Builds a personalized profile over time.
- **Dynamic Dashboard**: Real-time progress tracking for model retraining.
- **Multimodal Fusion**: Combines 4 different modalities for higher accuracy.
